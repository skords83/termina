## Design Context

This project has `PRODUCT.md` (register, users, brand personality, anti-references) and `DESIGN.md` (colors, typography, elevation, components) at the repo root. Read them before doing any UI/frontend design work — `impeccable` commands already do this automatically.

One-line summary: Termina is a self-hosted CalDAV calendar PWA for a single owner ("just me"), register `product`. Visual system: "The Warm Signal" — an almost-monochrome dark neutral field (ink-black → graphite) with exactly one saturated accent (Signal Blue `#4f7ef8`). Known tension to keep in mind: the current build reads as a precise dark dev-tool; the brand goal is "warm and personal, not clinical" — lean into that via hover states, motion, and copy rather than by adding a second accent color.
